python学习记录1

2015-10-20

Basic：

  1. 获取用户输入:input函数
    X=input(“x:”);
    Raw_input()的区别？
    Input会假设用户输入的是合法的python表达式，如果输入字符串没有问题，但不能强迫用户每次都输入引号，此时可以使用raw_input()
  2. Round()函数四舍五入为最接近的整数；
    Floor()函数向下取整；向上取整为ceil()函数
  3. 导入模块：
    （1） import math
    math.floor(32.9) 每次使用时都需要加math
    （2） from math import floor
    floor(32.9) 可以直接使用math中的函数
  4. #!加上程序的绝对路径，则可以直接执行python脚本，否则的话，执行脚本前需要加python，如：python test.py
  5. 注释：#
  6. 字符串拼接：
    （1） 直接一个接着另一个的方式写两个字符串，如：
    “hello” “zzt”
    用变量则不行，如：
    X=”hello”
    Y=”zzt”
    XY会输出语法错误
    （2） 上述情况使用X+Y
  7. 将值转换为字符串的两种机制：
    （1） str()函数
    （2） repr(x)函数，该函数可以使用反引号表示，如：x 将结果字符串转换为合法的python表达式
  8. 跨行的长字符串可以使用三个引号

列表和元组：

  1. 元组和列表的区别：统称为序列
    （1） 元组不能更改；
    （2） 表示方式：元组使用圆括号，列表使用方括号
  2. Python可以使用-1表示最后一个元素
  3. 序列的通用操作：
    （1） 索引：
    如果一个函数返回一个序列，那么可以直接对返回结果进行索引操作；
    （2） 分片：
    Num = [1,2,3,4,5,6,7,8,9,0]
    Num[3:6]输出[4,5,6]
    第一个索引是需要提取部分的第一个元素的编号，最后的索引是分片之后剩下部分的第一个元素编号；
    如果分片包括序列结尾的元素，只需置空最后一个索引即可；
    第三个参数是步长，默认为1，可以为负数，如：num[0:10:2]输出[1,3,5,7,9]，步长为负数时，即从右向左提取元素。
    （3） 相加和相乘
    （4） 空列表（[]）、创建一个占用十个元素空间，却不包含任何内容的列表，用none
    （5） 检查一个值是否在序列中，使用in运算符
  4. 列表的基本操作：
    （1） List函数可以根据字符串创建列表；
    相反，将列表转换为字符串，使用’’.join(somelist)
    （2） 赋值改变列表；x[1] = 2
    列表中删除元素：del x[1]
    分片赋值：x[2:5]=list(‘xxx’)，可以在不替换任何原有元素的情况下插入新的元素，当然也可以通过分片赋值来删除元素；
    （3） 列表方法：
    append():不是简单返回一个修改过的新列表，而是直接修改原来的列表
    count():统计某个元素在列表中出现的次数，如：x.count(1)
    extend():在一个列表末尾追加另一个序列的多个值，和连接操作(+)的区别:
    extend()修改了被扩展的序列，连接操作则返回一个新的列表
    index():找出某个值第一个匹配项的索引位置
    pop():移除列表中的一个元素，默认是最后一个，也可以用参数指定
    remove():移除列表中第一个匹配项
    reverse():列表中元素反向：返回一个迭代器对象，不是列表
    sort():对原列表排序，该方法返回值为空，如需要保留原先的副本，需要排序前先复制给另外一个列表
    列表复制的方法：
    Y=x 这样做将x和y都指向同一个列表
    Y=x[:] 这样复制了整个列表
    Sorted()函数不同，会返回排序好的列表，不改变原先的列表
    Sort()方法可以按照特定的方式进行排序，有另两个可选的参数（key,reverse），如key可以指定按照字符串的长度来排序，reverse指定列表是否需要反向排序
  5. 元组：
    （1） 实现一个值的元组，必须加一个逗号，如（42,）

字符串的使用：

  1. 字符串是不可以改变的
  2. 常用的字符串方法：
    Find()
    Join()/split()
    Lower()/upper()
    Replace()
    Strip()返回去除两侧空格的字符串
